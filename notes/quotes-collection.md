# Quotes Collection

Draft collection for a future public quotes page. Add quotes here in any order.
Keep one quote per entry so this can be converted into site data later.

## Format

```md
### Short working title

> Quote text.

Author: Name
Source: Book, essay, talk, interview, or unknown
URL: Optional source link
Tags: tag one, tag two
Notes: Optional private/context note, verification concern, or display idea
```

Minimum useful entry:

```md
### Short working title

> Quote text.

Author: Name
```

Use `unknown` when source or attribution needs verification. If a quote has line
breaks that matter, preserve them inside the blockquote.
